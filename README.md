# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/15-Eo5vFkjScjKDNz_5F3oUAQDLL3_pxB

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Open the app in your browser and click the Settings (⚙️) icon to enter your Gemini API key securely.
